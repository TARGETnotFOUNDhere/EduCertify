# EduCertify
🎓 EduCertify – Certificate Verification System

A web-based platform that allows users to verify academic certificates instantly using a unique Certificate ID.
EduCertify helps prevent certificate fraud and provides a simple digital verification system for institutions and employers.

📌 Project Overview

EduCertify is a certificate authentication platform built using HTML, CSS, and JavaScript.
The system verifies certificates by checking a Certificate ID against a stored database and displaying verified student information.

This project demonstrates how a simple web application can be used to improve trust, transparency, and efficiency in certificate verification.

🚀 Features

✅ Certificate verification using unique Certificate ID
✅ Verified certificate display with student details
✅ QR Code generation for quick verification
✅ Admin dashboard to add certificates
✅ Institution listing with subscription plans
✅ Simple and clean modern UI

⚙️ How It Works

User enters a Certificate ID on the verification page.

The system checks the ID against the certificate database.

If the certificate exists:

Student details are displayed.

A verified certificate badge appears.

A QR code is generated for quick verification.

If the ID is invalid, an error message is shown.
